Politique de
Confidentialité

Applicable à l'application Zeno Forme sur iOS.

Zeno Forme ne collecte aucune donnée personnelle. Toutes vos données restent stockées localement sur votre appareil.

Collecte de données

  • Aucune donnée personnelle n'est collectée
  • Toutes vos données restent sur votre téléphone :
    • Poids, pas, calories et autres données de santé
    • Notes personnelles et photos de progression
    • Objectifs et préférences

Utilisation des données

  • Aucune donnée n'est partagée, vendue ou transmise à des tiers
  • Les données sont utilisées uniquement dans l'application pour votre suivi personnel

Santé (HealthKit)

  • Zeno Forme accède à l'app Santé uniquement avec votre permission explicite
  • Les données sont synchronisées localement sur votre appareil
  • Aucune donnée de santé n'est envoyée vers des serveurs externes

Stockage local

  • Toutes les informations sont stockées dans la mémoire de votre téléphone
  • Vous pouvez supprimer toutes vos données à tout moment en désinstallant l'application

Internet

  • Zeno Forme n'utilise Internet que pour les mises à jour de l'application
  • Aucune donnée personnelle n'est transmise en ligne
