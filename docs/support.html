Questions fréquentes

Comment synchroniser mes données Santé ?

Zeno Forme se connecte automatiquement à l'app Santé d'Apple. Assurez-vous d'avoir autorisé l'accès lors du premier lancement. Vous pouvez modifier ces permissions dans Réglages > Santé > Accès aux données.

Mes données sont-elles sécurisées ?

Oui. Toutes vos données restent sur votre appareil et dans votre compte iCloud. Nous ne collectons ni ne vendons aucune donnée personnelle. Consultez notre politique de confidentialité pour plus d'informations.

L'application est-elle gratuite ?

Oui, Zeno Forme est actuellement entièrement gratuite. Certaines fonctionnalités premium pourront être ajoutées dans le futur.

Comment réinitialiser mon objectif calorique ?

Rendez-vous dans l'onglet "Bilan" puis "Profil". Vous pourrez modifier votre poids, taille et âge — l'application recalculera automatiquement vos objectifs.
