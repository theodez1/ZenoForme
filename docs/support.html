Questions fréquentes

Comment synchroniser mes données Santé ?

Zeno Forme se connecte automatiquement à l'app Santé d'Apple. Assurez-vous d'avoir autorisé l'accès lors du premier lancement de l'application. Vous pouvez modifier ces permissions dans Réglages > Santé > Accès aux données.

Mes données sont-elles sécurisées ?

Oui, toutes vos données restent sur votre appareil et dans votre compte iCloud. Nous ne collectons ni ne vendons aucune donnée personnelle. Consultez notre politique de confidentialité pour plus d'informations.

L'application est-elle gratuite ?

Oui, Zeno Forme est actuellement 100% gratuite. Certaines fonctionnalités premium pourront être ajoutées dans le futur.

Comment réinitialiser mon objectif calorique ?

Rendez-vous dans l'onglet "Bilan" puis "Profil". Vous pourrez modifier votre poids, taille, âge et l'app recalculera automatiquement vos objectifs.
